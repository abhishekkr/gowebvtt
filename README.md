## gowebvtt

> a Go package to parse **WebVTT subtitles** (Web Video Text Tracks Format) content 

* This package currently supports main features i.e. Time Marker, Subtitles & same-line Notes.

* To parse use `gowebvtt.ParseFile(..)`
